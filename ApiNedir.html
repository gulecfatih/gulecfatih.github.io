API Nedir?


API Nasıl Çalışır?

    İki ayrı uygulamanın birbiriyle konuşması için bir aracıya ihtiyacı vardır. Bu nedenle, geliştiriciler genellikle bir sistemin diğerinin bilgilerine veya fonksiyonlarına erişmesine izin vermek için köprüler — Uygulama Programlama Arayüzleri (API'ler) — oluşturur.

    Uygulamaları hızlı ve geniş ölçekte entegre etmek için API'ler, kablo üzerinden iletilen mesajların anlambilimini ve sözdizimini tanımlamak için protokoller ve/veya spesifikasyonlar kullanılarak gerçekleştirilir. Bu özellikler API mimarisini oluşturur.
